# Iris Species Classification using Machine Learning

This project predicts the species of an Iris flower using machine learning models.

The dataset contains measurements of iris flowers including:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

Three classification algorithms were implemented and compared:

• K-Nearest Neighbors (KNN)  
• Gaussian Naive Bayes  
• Support Vector Machine (SVM)

Steps performed:
- Data preprocessing
- Train-test split
- Model training
- Prediction
- Model evaluation using accuracy score and confusion matrix

Results:
KNN and SVM achieved 100% accuracy on the test dataset.

Tools used:
Python  
Pandas  
NumPy  
Scikit-learn
